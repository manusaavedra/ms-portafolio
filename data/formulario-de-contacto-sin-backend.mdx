Casi siempre que diseñamos un sitio web o portafolio y nos toca crear la sección de contacto
por lo general creamos un formulario que se conecta a un `endpoid` o que simplemente envia los datos a un servicio 
que se encarga de enviar la información que las persona que rellenen el formulario hayan puesto.

En este post te enseño una alternativa para formularios de contactos en tu portafolio o sitio web sin necesidad 
de backend ni servicios externos.

## ¿Qué es un enlace mailto en HTML?

Los enlaces mailto se utilizan para redirigir al usuario a una dirección de correo electrónico en lugar de un enlace. 
Cuando el usuario hace clic en un vínculo mailto, el cliente de correo electrónico predeterminado se abre en el equipo 
del usuario y sugiere enviar un correo electrónico a la dirección incluida en el vínculo mailto.

## ¿Cómo crear enlaces mailto en HTML?
Puede crear un vínculo mailto HTML utilizando una etiqueta de anclaje con el atributo 
href e insertando el parámetro "mailto" después de él.

`<a href="mailito:me@gmail.com"></a>`

Sabiendo esto aprovecharemos esta alternativa y crearemos un formulario y lo haremos un poco más dinámico

```html:/index.html

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>Document</title>
	<link rel="stylesheet" href="./style.css" />
</head>
<body>
	<form id="form">
		<div class="form-inputs">
			<input name="names" type="text" placeholder="names">
		</div>
		<div class="form-inputs">
			<input name="email" type="email" placeholder="email">
		</div>
		<div class="form-inputs">
			<textarea name="message" placeholder="message" cols="30" rows="10"></textarea>
		</div>
		<div class="form-inputs">
			<a id="triggerMail"></a>
			<button name="email" type="email" placeholder="email">
				Enviar Email
			</button>
		</div>
	</form>
	<script src="./main.js" type="text/javascript"></script>
</body>
</html>
```

Le damos un poco de estilos al documento html. 💅🥲

```css:/style.css 

body {
	display: grid;
	place-items: center;
	height: 100vh;
}

#triggerMail {
  	opacity: 0;
}

.form-inputs {
  	margin: 4px 0;
}

input,
textarea {
	padding: 4px 15px;
	border-radius: 6px;
	border: 1px solid #e6e6e6;
	resize: none;
	width: 100%;
}

button {
	padding: 8px;
	width: 100%;
	font-size: 16px;
	background-color: cornflowerblue;
	border: 1px solid cornflowerblue;
	border-radius: 6px;
	color: #fff;
}
```

ahora vamos a javascript 👇

```js:/main.js  

const $ = (selector) => document.querySelector(selector) 
$('#form').addEventListener('submit', handleSubmit)

function handleSubmit(event) {
	event.preventDefault()
	const data = new FormData(this)

	const subject = `${data.get("names")}, Saludos!`
	const body = `${data.get("names")}:${data.get("email")}\n ${data.get("message")}`

	$("#triggerMail").setAttribute("href", `mailito:manusaavedra.portafolioweb@gmail.com?subject${subject}=&body=${body}`)

	console.log(body)
	$("#triggerMail").click()
}

```

Basicamente lo que hemos hecho es obtener los datos del formulario `<form id="form">` y 
utilizar el elemento `<a id="triggerMail"></a>` y con javascript añadimos su atributo 
`href` con el valor de nuestro enlace `mailito` con los datos del formulario.
